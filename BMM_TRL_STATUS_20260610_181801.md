# BMM-TRL Status - 20260610_181801

## Current Status

The latest diagnostics changed the interpretation of the PointMaze high-budget
failure.

The BMM-TRL implementation is probably not the primary blocker. The strongest
current conclusion is:

```text
For PointMaze medium, high-budget same-trajectory logged-offset labels are
behavior-time labels, not clean reachability labels.
```

After adding graph and layout/grid geodesic diagnostics, `H=256` and `H=512`
are beyond the calibrated maze diameter for PointMaze medium. A clean
maze-geodesic reachability target should be all-positive at those budgets. The
previous high-budget negatives came from behavior-policy detours, not from true
unreachability.

## Evidence Before Geodesic Target

Earlier diagnostics showed:

1. The same JAX BMM critic path learns a deterministic neural chain through
   `H=512`.
2. PointMaze fixed-batch overfit succeeds through `H=512` in both state-only
   and action-conditioned modes.
3. PointMaze heldout logged-offset labels fail at `H=256/512`.
4. Euclidean and action-goal baselines are near chance at `H=256/512`.
5. kNN on logged-offset labels is also near chance at high budgets:
   - `H=256`: best kNN AUC about `0.54`
   - `H=512`: best kNN AUC about `0.52`

This means logged offset is not identifiable from `(s, g, H)` or
`(s, a, g, H)` at high budgets.

## Recommended Next Steps

1. Stop using high-budget logged-offset negatives as the main PointMaze
   reachability gate.
2. For PointMaze medium, run geodesic diagnostics on meaningful budgets such as
   `32,64,96,128,160`; `256/512` are expected one-class positives.
3. Add a graph/grid-label training path for more than one frozen batch before
   returning to BMM max-min consistency.
4. For action-conditioned diagnostics, label
   `Q_H(s,a,g) = 1[d(s_next, g) <= H - 1]`.
5. If high-budget binary separation is required, move to a larger maze whose
   calibrated geodesic diameter exceeds `256/512`, or change the budget scale.
6. Only after graph/grid supervised critics pass should BMM transitive
   consistency and policy-facing evaluation be re-enabled.
